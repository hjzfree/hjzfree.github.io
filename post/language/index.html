language

2020-11-14 1 min read

这几天看到一篇文章讲崔琦的。文章中提到崔琦说，华人研读科学应该中英文交错使用，才可兼容并蓄，收到真正学习之效。只懂得中文会令科学家无法追读到最新的科研报告，而完全放弃中文却是舍本逐末。
后面这个舍本逐末让我感触比较深。我最近试图完全使用英文来维护这个博客，可是发现很难表达出自己的想法，搞得自己郁郁不乐。
今天刚刚安装上谷歌拼音，特意来试一试。可能今后的一段时间确实是要中英夹杂了。或许等一段时间又会因为强迫症而把这删掉。但总是希望过程比结果要重要吧。

sudo apt-get install fcitx-googlepinyin
language support ---> iBus to fctix
Configure ---> + ---> cancel Only Show Current Language ---> Google Pinyin
ctrl + space
